Abilities:
Enhanced phsyiology, Electrokinesis, Acoustokinesis (Via "Paleo Memory Bytes"), Teleportation/Time Manipulation (Via "Chaos Memory Bytes"), Telekinesis (Via "Magic Memory Bytes")
  • Name: Enhanced Physiology
  • Description: Zoe's body is enhanced, allowing her to perform feats at a supernatural level
  • Source: Aether infusion
  • Drawback: While her body is more enhanced, she still has fundamental limits

  • Name: Acoustokinesis
  • Description: from collecting the Paleo Memory Bytes, Zoe is able to generate powerful soundwaves, capable of causing damage or shattering glass
  • Source: Paleo Memory Bytes
  • Drawback: The soundwaves can be absorbed by certain materials, and due to the nature of the Memory Bytes, She sees visions of the memories the bytes are formed from

  • Name: Teleportation/Time Manipulation
  • Description: From collecting the Chaos Memory Bytes, Zoe is able to briefly stop time, as well as teleport a short distance
  • Source: Chaos Memory Bytes
  • Drawback: Both the time stop and the teleportation take time to recharge, as well as having the same memory visions as the other memory bytes

  • Name: Telekinesis
  • Description: From collecting the Magic Memory Bytes, Zoe is able to levitate objects with her mind, controlled objects having a purple aura
  • Source: Magic Memory Bytes
  • Drawback: While she is able to lift many objects, heavier objects take more effort. She also has the same memory visions as the other memory bytes

  • Name: Electrokinesis
  • Description: A power deep within Zoe, She is able to control and manipulate electricity
  • Source: Aether infusion (dormant)
  • Drawback: Absorbing or controlling too much electricity will overload Zoe. Accessing the full extent of her power also renders the memory bytes unusable while using such power
